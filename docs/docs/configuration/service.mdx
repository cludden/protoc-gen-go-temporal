# Service

Service options apply to all Temporal resources defined within an individual protobuf service.

```protobuf title="example.proto"
syntax="proto3";

package example.v1;

import "temporal/v1/temporal.proto";

service Example {
  option (temporal.v1.service) = {
    task_queue: "example-v1"
  };
}
```

## Options

### task_queue

`string`

Specifies the default [Task Queue](https://docs.temporal.io/workers#task-queue) name.

```protobuf
service Example {
  option (temporal.v1.service) = {
    task_queue: 'example-v1'
  };
}
```
